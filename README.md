# Assistent de Problemes Informàtics amb IA

Aquest projecte és un assistent intel·ligent dissenyat per ajudar a resoldre problemes tècnics comuns mitjançant l'ús d'IA generativa local.

## Característiques principals
- **IA Local:** Utilitza Ollama per garantir la privacitat de les dades.
- **Diagnòstic Agèntic:** L'IA guia l'usuari a través d'un procés de resolució pas a pas.
- **Tecnologies:** Vue.js, Ollama, Node.js.

## Estructura del projecte
- `src/`: Codi font de l'aplicació web.
- `docs/`: Documentació, informes i presentació.
- `PROCES.md`: Registre detallat del desenvolupament (per ús intern).
