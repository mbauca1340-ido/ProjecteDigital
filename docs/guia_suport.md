# Guia de Resolució de Problemes Comuns - Suport Tècnic

## 1. Problemes de Pantalla
- **Pantalla negra:** Comprovar el cable d'alimentació i el cable de vídeo (HDMI/DisplayPort). Si és un portàtil, provar de connectar una pantalla externa.
- **Línies a la pantalla:** Pot ser un problema de la targeta gràfica o del cable. Reiniciar i entrar a la BIOS per veure si les línies persisteixen.

## 2. Problemes de Connexió (Xarxa)
- **Sense Internet:** Reiniciar el router. Comprovar si el mode avió està activat. Executar `ipconfig /flushdns` a la consola de Windows.
- **Wi-Fi lent:** Apropar-se al router o utilitzar un cable Ethernet. Evitar interferències de microones o telèfons sense fils.

## 3. Rendiment del Sistema
- **Ordinador lent:** Obrir l'Administrador de Tasques i tancar processos que consumeixen molta CPU o RAM. Comprovar l'espai lliure al disc dur (mínim 10-15%).
- **Sobrecalentament:** Netejar les sortides d'aire. No utilitzar el portàtil sobre superfícies toves com el llit.

## 4. Impressores
- **No imprimeix:** Comprovar la cua d'impressió. Reiniciar el servei de "Cola de impresión" (spooler) a Windows.
- **Fulls encallats:** Retirar el paper amb cura seguint la direcció del mecanisme, mai estirant fort cap enrere.

## 5. Software i Virus
- **Aplicació no respon:** Forçar la finalització des de l'Administrador de Tasques (Ctrl+Alt+Supr).
- **Sospita de virus:** Executar una anàlisi completa amb Windows Defender o un antivirus de confiança. No obrir fitxers .exe adjunts en correus desconeguts.
